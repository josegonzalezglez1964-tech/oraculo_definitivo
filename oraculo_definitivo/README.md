# Suerte Canaria | Aplicación Fullstack 🌊✨

**Suerte Canaria** es una plataforma interactiva diseñada para fusionar la estadística de juegos de azar con la mística climática y astral de la isla de El Hierro. Es un producto de enfocado en ofrecer una experiencia de usuario inmersiva bajo una estética oscura y moderna.

---

## 🚀 Características Principales

### 1. El Oráculo (Generador Místico)
Permite generar combinaciones numéricas basadas en diferentes "fuentes de energía":
* **Alisios + Astros**: Fusión completa de datos meteorológicos y vibración astral.
* **Clima de El Hierro**: Utiliza datos específicos como el viento de Gorona del Viento y el calor de El Pinar.
* **Horóscopo**: Integra la energía del signo zodiacal del usuario.
* **Juegos compatibles**: Configurado para generar números para **SúperOnce** **Triple** **Mi dia** **Dupla**.

### 2. Análisis de Racha (Estadística)
Visualización técnica de los números en los sorteos de la ONCE:
* **Números Calientes**: Identificación de los números con mayor frecuencia de aparición (ej. el 12 con un 4.7%).
* **Números Fríos**: Listado de números con menor actividad reciente.
* **Historial**: Registro de los últimos sorteos realizados entre abril y mayo de 2026.

### 3. Tira de Clima en Tiempo Real
Panel informativo con datos ambientales de El Hierro que influyen en la "magia" del sitio:
* **El Pinar**: Temperatura (21,3 °C) y humedad (82%).
* **Gorona del Viento**: Velocidad y dirección del viento (35,9 km/h NNE).
* **Atlántico**: Altura de las olas (3 metros).
* **Alisios**: Estado de actividad de los vientos dominantes.

### 4. Tu Bitácora
Sección personalizada donde los usuarios pueden guardar y consultar sus combinaciones favoritas generadas previamente.

## 🛠️ Stack Tecnológico

* **Framework de Estilos**: Tailwind CSS para un diseño responsivo y modo oscuro nativo.
* **Iconografía**: Lucide Icons para la representación visual de elementos climáticos y de navegación.
* **Tipografías**: Uso de fuentes Display para títulos, Inter para el cuerpo y fuentes Mono para datos estadísticos.
* **Efectos Visuales**: Implementación de efectos de grano (`grain`), desenfoques de fondo (`backdrop-blur`) y animaciones de brillo pulsante (`pulse-glow`).
